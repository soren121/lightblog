If you encounter any bugs, please be courteous and
report them at
http://code.google.com/p/lightblog/issues/entry

Installation:

1) Download a copy of the SVN.
2) Place the files where you want in your web
   server.
3) Navigate to the directory you placed
   LightBlog in in your web browser.
4) LightBlog will begin the installer.
5) Follow the steps as instructed by the installer.
6) You're finished! Just remember, REMOVE install.php!
